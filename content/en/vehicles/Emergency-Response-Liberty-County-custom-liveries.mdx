<Section icon={Target} title="Step-by-Step Creation and Upload Process">

Mastering the workflow for **Emergency Response Liberty County custom liveries** requires precision. One small mistake in the Asset ID or the upload category can result in your livery not appearing correctly in the game environment.

<Steps>
  <Step num="1" title="Download Official Templates">
    Navigate to the PRC Discord server and locate the **Bulletin Board**. Click the link for the custom livery packs to access the Google Drive or MEGA folder containing the latest vehicle templates.
  </Step>
  <Step num="2" title="Design Your Graphics">
    Open your chosen template (e.g., the Right Side of the Bullhorn Prer Pursuit) in your editor. Add your department name, unit numbers, and badges. **Do not** change the base vehicle color in the editor if you want to use the in-game color picker later.
  </Step>
  <Step num="3" title="Upload to Roblox Creator Dashboard">
    Go to the Roblox website, click **Create**, then navigate to **Development Items** > **Decals**. Upload your PNG file here. Once uploaded, click on the item to open its page and copy the numerical **Asset ID** from the URL.
  </Step>
  <Step num="4" title="Configure in ERLC Server Settings">
    Join your private server, open the **Server Settings**, and navigate to the **Livery Pack** section. Find the vehicle model you designed for, click **Add New**, and paste the Asset ID into the corresponding slot (e.g., Right Side).
  </Step>
</Steps>

| Upload Step | Action Required | Common Error |
| :--- | :--- | :--- |
| **Asset Type** | Select **Decal** in Creator Dashboard | Selecting "Image" or "Shirt" by mistake |
| **Naming** | Use a clear, descriptive name | Using banned words that trigger Roblox filters |
| **ID Extraction** | Copy only the digits from the URL | Including slashes or extra characters |
| **Placement** | Match the ID to the correct vehicle side | Pasting the "Left" ID into the "Right" slot |

<Section icon={Shield} title="Color Matching and Transparency Techniques">

A common issue with **Emergency Response Liberty County custom liveries** is color mismatching between the custom decal and the vehicle's base paint. To achieve a seamless look, you must understand how the game handles Hex codes.

If you want the vehicle color and the livery color to be identical, follow these steps:
1. Select your desired vehicle color in the ERLC Livery menu.
2. Copy the **Hex Code** (e.g., #FFFFFF for white).
3. Paste that same Hex Code into the **Livery Color** field located directly below the Asset ID slots.

Alternatively, you can create **Transparent Liveries**. By deleting the background layer in your design software and only exporting the text and logos, the vehicle's base paint will show through your design. This is the most flexible method for server owners who want to offer multiple color options for the same department.

<Section icon={Trophy} title="Submission, Approval, and Guidelines">

Once your livery is configured in the server settings, it must be submitted for approval by the **Emergency Response Liberty County** moderation team. This is a manual check to ensure the content is appropriate for the game's community.

| Status Icon | Meaning | Expected Action |
| :--- | :--- | :--- |
| **White Timer** | Under Review | Wait 24-72 hours for a moderator to check the file |
| **Green Check** | Approved | The livery is now visible to all players in your server |
| **Missing/Removed** | Declined | Check the guidelines and re-upload a corrected version |

**Common Rejection Reasons:**
- **Copyrighted Logos**: Using real-world brand logos (like Ford, Chevy, or specific real-life police badges) can lead to a decline. It is safer to use generic or "County/State" style logos.
- **External Links**: Do not include Discord invites, social media handles, or website URLs on the livery.
- **Inappropriate Content**: Any offensive language or symbols will result in an immediate decline and potential server warnings.

<Callout type="info" title="Approval Times">
Approval times are inconsistent. While some liveries are accepted within 2 hours, others may take up to 3 days depending on the volume of submissions. If your livery is still "Under Review" after 4 days, consider re-submitting.
</Callout>
