<Section icon={Sparkles} title="Private Server Management & Custom Liveries">

Private servers are highly popular for organized roleplay communities. Recent updates in the **Emergency Response Liberty County update log** have significantly expanded what server owners can customize.

### Custom Livery and Advertisement Guidelines
If your server has the Custom Livery Pack, you can upload custom vehicle designs. However, all submissions must pass a strict review process:

- **Template Accuracy**: You must download and use the official template for the specific vehicle model.
- **No External Links**: QR codes, social media handles, website links, or phone numbers are strictly prohibited.
- **Readability**: Text and logos must be large, high-contrast, and clearly legible.
- **Real-World Agencies**: Designs based on real-world city, county, or state emergency services are permitted as long as they do not violate copyright or contain prohibited text.
- **Server Branding**: Staff, Moderator, or Admin liveries must clearly display the name or abbreviation of your specific private server.

### Private Server API Features (v2)
For communities running external CAD/MDT systems, the February 2026 API expansion provides crucial data points:

- **Live Coordinates**: Track the exact X, Y, Z positions of active players for live-mapping.
- **Vehicle Customization Tracking**: Pull vehicle color data directly into your community dispatch panels.
- **Secure Integration**: Access your unique API key securely through the in-game Private Server Settings menu.
