<Section icon={Target} title="Schritt-für-Schritt-Erstellung und Upload-Prozess">

Die Beherrschung des Arbeitsablaufs für **eigene Lackierungen in Emergency Response Liberty County** erfordert Präzision. Ein kleiner Fehler bei der Asset-ID oder der Upload-Kategorie kann dazu führen, dass Ihre Lackierung in der Spielumgebung nicht korrekt erscheint.

<Steps>
  <Step num="1" title="Offizielle Vorlagen herunterladen">
    Navigieren Sie zum PRC Discord-Server und suchen Sie das **Bulletin Board**. Klicken Sie auf den Link für die benutzerdefinierten Lackierungspakete, um auf den Google Drive- oder MEGA-Ordner mit den neuesten Fahrzeugvorlagen zuzugreifen.
  </Step>
  <Step num="2" title="Grafiken entwerfen">
    Öffnen Sie die gewählte Vorlage (z. B. die rechte Seite des Bullhorn Prer Pursuit) in Ihrem Editor. Fügen Sie den Namen Ihrer Behörde, Einheitennummern und Abzeichen hinzu. Ändern Sie die Basisfarbe des Fahrzeugs **nicht** im Editor, wenn Sie später die Farbauswahl im Spiel verwenden möchten.
  </Step>
  <Step num="3" title="Im Roblox Creator Dashboard hochladen">
    Gehen Sie auf die Roblox-Website, klicken Sie auf **Erstellen** und navigieren Sie dann zu **Entwicklungselemente** > **Decals**. Laden Sie hier Ihre PNG-Datei hoch. Klicken Sie nach dem Hochladen auf das Element, um dessen Seite zu öffnen, und kopieren Sie die numerische **Asset-ID** aus der URL.
  </Step>
  <Step num="4" title="In den ERLC-Servereinstellungen konfigurieren">
    Treten Sie Ihrem privaten Server bei, öffnen Sie die **Servereinstellungen** und navigieren Sie zum Abschnitt **Livery Pack**. Suchen Sie das Fahrzeugmodell, für das Sie das Design erstellt haben, klicken Sie auf **Neu hinzufügen** und fügen Sie die Asset-ID in den entsprechenden Slot ein (z. B. Rechte Seite).
  </Step>
</Steps>

| Upload-Schritt | Erforderliche Aktion | Häufiger Fehler |
| :--- | :--- | :--- |
| **Asset-Typ** | Wählen Sie **Decal** im Creator Dashboard | Auswahl von „Image“ oder „Shirt“ aus Versehen |
| **Benennung** | Verwenden Sie einen klaren Namen | Verwendung gesperrter Wörter, die Roblox-Filter auslösen |
| **ID-Extraktion** | Kopieren Sie nur die Ziffern aus der URL | Einschließen von Schrägstrichen oder Zusatzzeichen |
| **Platzierung** | ID dem korrekten Fahrzeugteil zuordnen | Einfügen der „Links“-ID in den „Rechts“-Slot |

<Callout type="warning" title="Asset-ID-Verarbeitung">
Nach dem Hochladen auf Roblox kann das Bild für einige Minuten als „beschädigtes“ Symbol erscheinen. Das ist normal; Roblox scannt das Bild auf Sicherheit. Sie können die ID trotzdem kopieren, aber sie wird in ERLC möglicherweise erst angezeigt, wenn der Scan abgeschlossen ist.
</Callout>

<Section icon={Shield} title="Farbanpassung und Transparenztechniken">

Ein häufiges Problem bei **eigenen Lackierungen für Emergency Response Liberty County** ist die farbliche Abweichung zwischen dem benutzerdefinierten Decal und dem Basislack des Fahrzeugs. Um einen nahtlosen Look zu erzielen, müssen Sie verstehen, wie das Spiel mit Hex-Codes umgeht.

Wenn die Fahrzeugfarbe und die Lackierungsfarbe identisch sein sollen, folgen Sie diesen Schritten:
1. Wählen Sie Ihre gewünschte Fahrzeugfarbe im ERLC-Lackierungsmenü aus.
2. Kopieren Sie den **Hex-Code** (z. B. #FFFFFF für Weiß).
3. Fügen Sie denselben Hex-Code in das Feld **Lackierungsfarbe** direkt unter den Asset-ID-Slots ein.

Alternativ können Sie **transparente Lackierungen** erstellen. Indem Sie die Hintergrundebene in Ihrer Designsoftware löschen und nur Text und Logos exportieren, scheint der Basislack des Fahrzeugs durch Ihr Design hindurch. Dies ist die flexibelste Methode für Serverbesitzer, die mehrere Farboptionen für dieselbe Behörde anbieten möchten.

<Section icon={Trophy} title="Einreichung, Genehmigung und Richtlinien">

Sobald Ihre Lackierung in den Servereinstellungen konfiguriert ist, muss sie dem Moderationsteam von **Emergency Response Liberty County** zur Genehmigung vorgelegt werden. Dies ist eine manuelle Prüfung, um sicherzustellen, dass der Inhalt für die Community angemessen ist.

| Statussymbol | Bedeutung | Erwartete Aktion |
| :--- | :--- | :--- |
| **Weißer Timer** | In Prüfung | Warten Sie 24–72 Stunden auf die Überprüfung durch einen Moderator |
| **Grüner Haken** | Genehmigt | Die Lackierung ist nun für alle Spieler auf Ihrem Server sichtbar |
| **Fehlt/Entfernt** | Abgelehnt | Prüfen Sie die Richtlinien und laden Sie eine korrigierte Version hoch |

**Häufige Ablehnungsgründe:**
- **Urheberrechtlich geschützte Logos**: Die Verwendung realer Markenlogos (wie Ford, Chevy oder spezifische echte Polizeiabzeichen) kann zur Ablehnung führen. Es ist sicherer, generische Logos oder Logos im „County/State“-Stil zu verwenden.
- **Externe Links**: Fügen Sie keine Discord-Einladungen, Social-Media-Handles oder Website-URLs auf der Lackierung ein.
- **Unangemessene Inhalte**: Jegliche beleidigende Sprache oder Symbole führen zu einer sofortigen Ablehnung und potenziellen Server-Warnungen.

<Callout type="info" title="Genehmigungszeiten">
Die Genehmigungszeiten sind unterschiedlich. Während einige Lackierungen innerhalb von 2 Stunden akzeptiert werden, können andere je nach Aufkommen bis zu 3 Tage dauern. Wenn Ihre Lackierung nach 4 Tagen immer noch „In Prüfung“ ist, ziehen Sie eine erneute Einreichung in Betracht.
</Callout>

</Section>
