<Section icon={Target} title="Passo a Passo de Criação e Envio">

Dominar o fluxo de trabalho para os **adesivos personalizados no Emergency Response Liberty County** exige precisão. Um pequeno erro no ID de Ativo ou na categoria de envio pode fazer com que seu adesivo não apareça corretamente no ambiente do jogo.

<Steps>
  <Step num="1" title="Baixe os Templates Oficiais">
    Acesse o servidor do Discord da PRC e localize o canal **Bulletin Board** (Painel de Anúncios). Clique no link dos pacotes de adesivos personalizados para acessar a pasta do Google Drive ou MEGA que contém os templates de veículos mais recentes.
  </Step>
  <Step num="2" title="Desenhe seus Gráficos">
    Abra o template escolhido (por exemplo, a lateral direita do Bullhorn Prer Pursuit) no seu editor de imagem. Adicione o nome do seu departamento, números de unidade e distintivos. **Não** altere a cor de base do veículo no editor se quiser usar o seletor de cores do jogo mais tarde.
  </Step>
  <Step num="3" title="Envie para o Painel do Criador do Roblox">
    Acesse o site do Roblox, clique em **Criar**, depois navegue até **Itens de Desenvolvimento** > **Decalques**. Envie seu arquivo PNG aqui. Após o envio, clique no item para abrir a página dele e copie o **ID de Ativo (Asset ID)** numérico da URL.
  </Step>
  <Step num="4" title="Configure nas Opções do Servidor no ERLC">
    Entre no seu servidor privado, abra as **Configurações do Servidor** e navegue até a seção **Livery Pack**. Encontre o modelo de veículo para o qual você desenhou, clique em **Adicionar Novo** e cole o ID de Ativo no campo correspondente (ex: Lateral Direita).
  </Step>
</Steps>

| Passo de Envio | Ação Necessária | Erro Comum |
| :--- | :--- | :--- |
| **Tipo de Ativo** | Selecionar **Decalque (Decal)** no Painel do Criador | Selecionar \"Imagem\" ou \"Camisa\" por engano |
| **Nomeação** | Usar um nome claro e descritivo | Usar palavras banidas que ativam os filtros do Roblox |
| **Extração de ID** | Copiar apenas os números da URL | Incluir barras ou caracteres adicionais |
| **Posicionamento** | Associar o ID ao lado correto do veículo | Colar o ID do lado \"Esquerdo\" no campo do lado \"Direito\" |

<Callout type="warning" title="Processamento do ID de Ativo">
Após o envio para o Roblox, a imagem pode aparecer com um ícone de \"quebrada\" por alguns minutos. Isso é normal; o Roblox está analisando a imagem por questões de segurança. Você ainda pode copiar o ID, mas ele pode não aparecer no ERLC até que a verificação seja concluída.
</Callout>

<Section icon={Shield} title="Técnicas de Combinação de Cores e Transparência">

Um problema comum com os **adesivos personalizados no Emergency Response Liberty County** é a diferença de cor entre o decalque personalizado e a pintura de base do veículo. Para obter um visual perfeito, você precisa entender como o jogo lida com códigos Hex.

Se você deseja que a cor do veículo e a cor do adesivo sejam idênticas, siga estes passos:
1. Selecione a cor desejada para o veículo no menu de Adesivos do ERLC.
2. Copie o **Código Hex** (ex: #FFFFFF para branco).
3. Cole esse mesmo Código Hex no campo **Livery Color** localizado logo abaixo dos campos de ID de Ativo.

Alternativamente, você pode criar **Adesivos Transparentes**. Ao deletar a camada de fundo no seu software de design e exportar apenas o texto e os logotipos, a pintura de base do veículo aparecerá por trás do seu design. Este é o método mais flexível para proprietários de servidores que desejam oferecer várias opções de cores para o mesmo departamento.

<Section icon={Trophy} title="Envio, Aprovação e Diretrizes">

Depois que o seu adesivo estiver configurado nas opções do servidor, ele deve ser enviado para aprovação pela equipe de moderação do **Emergency Response Liberty County**. Esta é uma verificação manual para garantir que o conteúdo seja apropriado para a comunidade do jogo.

| Ícone de Status | Significado | Ação Esperada |
| :--- | :--- | :--- |
| **Cronômetro Branco** | Em Revisão | Aguarde de 24 a 72 horas para que um moderador verifique o arquivo |
| **Check Verde** | Aprovado | O adesivo agora está visível para todos os jogadores no seu servidor |
| **Ausente/Removido** | Recusado | Verifique as diretrizes e envie novamente uma versão corrigida |

**Motivos Comuns de Rejeição:**
- **Logotipos Protegidos por Direitos Autorais**: O uso de logotipos de marcas do mundo real (como Ford, Chevrolet ou distintivos de polícias reais específicas) pode levar à rejeição. É mais seguro usar logotipos genéricos ou no estilo \"Condado/Estado\".
- **Links Externos**: Não inclua convites do Discord, redes sociais ou URLs de sites no adesivo.
- **Conteúdo Inadequado**: Qualquer linguagem ou símbolo ofensivo resultará em uma rejeição imediata e possíveis advertências ao servidor.

<Callout type="info" title="Prazos de Aprovação">
Os tempos de aprovação variam. Enquanto alguns adesivos são aceitos em até 2 horas, outros podem levar até 3 dias, dependendo do volume de envios. Se o seu adesivo ainda estiver \"Em Revisão\" após 4 dias, considere enviá-lo novamente.
</Callout>
