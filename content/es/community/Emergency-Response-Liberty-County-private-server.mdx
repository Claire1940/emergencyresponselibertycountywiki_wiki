<Section icon={Shield} title="Introducción a la Configuración del Servidor Privado de Emergency Response Liberty County">

Un **servidor privado de Emergency Response Liberty County** ofrece a las comunidades un entorno de sandbox dedicado para organizar patrullas personalizadas, entrenamientos de departamentos y eventos de rol estructurados. A diferencia de los servidores públicos, los propietarios de servidores privados tienen total autoridad sobre las configuraciones del juego, los permisos del personal y los elementos personalizados. 

Para maximizar el potencial de tu comunidad personalizada, debes comprender cómo interactúa la propiedad del servidor con los paquetes premium, como el Paquete de Diseños Personalizados (Custom Livery Pack) y el Paquete de API (API Pack). Gestionar estas funciones correctamente garantiza un entorno estable, atractivo y profesional para tus jugadores.

| Paquete de Funciones | Función Principal | Requisito |
|---|---|---|
| **Servidor Privado Base** | Alojar sesiones privadas, gestionar configuraciones básicas, expulsar/banear jugadores | Compra con Robux |
| **Paquete de Diseños Personalizados** | Subir aspectos de vehículos personalizados y uniformes de departamento | Gamepass por separado |
| **Paquete de Servidor API** | Conectar bots de Discord externos y recuperar datos del servidor en tiempo real | Gamepass por separado |

| Rol del Personal | Acciones Permitidas | Nivel de Responsabilidad |

<Section icon={Target} title="Guía Paso a Paso para Subir Diseños Personalizados">

Los diseños personalizados (liveries) permiten que tu comunidad use diseños de vehículos únicos basados en agencias del mundo real o departamentos ficticios. Para usar esta función, tu servidor debe tener activo el Paquete de Diseños Personalizados y todos los diseños deben pasar el proceso de revisión oficial.

<Steps>
  <Step num="1" title="Descargar Plantillas Oficiales">
    Visita los recursos oficiales de los desarrolladores para descargar la plantilla de imagen precisa para el vehículo específico que deseas personalizar.
  </Step>
  <Step num="2" title="Aplicar Gráficos Personalizados">
    Usa un software de edición de imágenes para colocar las marcas, logotipos y capas de texto de alto contraste de tu departamento directamente en la plantilla.
  </Step>
  <Step num="3" title="Subir a Roblox">
    Sube tu plantilla completada como un elemento Decal a través del Panel del Creador de Roblox y guarda el ID de Elemento (Asset ID) único.
  </Step>
  <Step num="4" title="Enviar para Revisión en el Juego">
    Abre el editor de diseños dentro de tu servidor privado de Emergency Response Liberty County, ingresa el ID de Elemento y envíalo para su revisión oficial.
  </Step>
</Steps>

<Callout type="success" title="Consejo para la Aprobación de Diseños">
Asegúrate de que todo el texto de tu diseño sea claramente legible y tenga un alto contraste con el color de fondo para evitar el rechazo durante el proceso de revisión.
</Callout>

</Section>

<Section icon={Sparkles} title="Reglas de Cumplimiento de Diseños y Publicidad">

Todo el contenido personalizado subido a tu servidor debe cumplir con los Términos de Servicio de Roblox y las directrices específicas establecidas por el equipo de desarrollo de Police Roleplay Community.

| Tipo de Contenido | Elementos Permitidos | Elementos Prohibidos |
|---|---|---|
| **Diseños de Vehículos** | Diseños ficticios, réplicas de agencias del mundo real, números de emergencia estándar (911/999) | Códigos QR, enlaces a redes sociales, nombres personales, infracciones de derechos de autor |
| **Anuncios en Carteles (16:9)** | Logotipos del servidor, nombres de la comunidad, capturas de pantalla del juego de alta calidad | Enlaces de Discord, sorteos de Robux, anuncios de reclutamiento de personal |
| **Anuncios en Paradas de Autobús (2:3)** | Abreviaturas del servidor, ilustraciones de rol limpias, horarios de eventos | Enlaces de comunicación externa, memes de baja calidad |

<Callout type="info" title="Optimización de Publicidad">
Mantén limpios los diseños de los carteles y de las paradas de autobús. Las directrices oficiales recomiendan una resolución máxima de 512 píxeles en cualquier eje para mantener un rendimiento óptimo.
</Callout>

Para comunidades avanzadas, el Paquete de Servidor API te permite conectar aplicaciones externas a tu servidor privado de Emergency Response Liberty County. Esto permite la moderación automatizada, el seguimiento de jugadores en tiempo real y las integraciones personalizadas de bots de Discord.
